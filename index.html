ClinPhoDoc

A React Native app for clinicians who need to capture clinical photos without relying on insecure camera-roll workflows. The product is designed around a real bedside problem: photos are often taken before charting is complete, but default phone storage and cloud syncing create obvious privacy and workflow issues.

React Native
Expo
TypeScript
Healthcare workflow
Secure local photo storage
Reminder-based follow-up
Clinical usability
  • Built for the specific workflow where clinicians photograph ventilator or treatment settings at bedside and complete documentation later.
  • Keeps photos inside app-managed local storage rather than a general-purpose photo library, reducing the risk of accidental cloud sync or over-broad app access.
  • Adds reminder and to-do behavior so captured photo sets can be tracked until they have been charted on, making the app useful beyond simple image capture.

MyMovie List

An Android app focused on responsive movie browsing and reliable offline behavior. The project uses modern Android architecture to keep API-driven UI state coherent across nested navigation and asynchronous data flow.

Kotlin
Jetpack Compose
Room
MVVM
Reactive UI
Offline caching
StateFlow
  • Engineered the UI with Jetpack Compose and MVVM so nested navigation flows and async API state stay synchronized cleanly.
  • Used shared ViewModel scoping and StateFlow to coordinate updates across screens instead of letting state drift between navigation contexts.
  • Added a Room-backed caching layer with custom type converters alongside Retrofit and coroutines to improve performance and make the app more resilient offline.

CommunityPulse

Built for Hack the Globe 2026, CommunityPulse helps outreach workers capture interviews, anonymize sensitive information, analyze social determinants of health, and visualize community needs on a geographic heatmap.

Next.js
TypeScript
Leaflet
Ollama
Interview transcription
Privacy-first analysis
Geographic heatmap
  • Uses a two-stage local-LLM pipeline: first to anonymize transcripts and generalize location data, then to extract structured health and socioeconomic signals.
  • Plots analyzed encounter data onto an interactive map so health and policy teams can explore concentrated needs by age, gender, and other filters.
  • Combines speech capture, structured analysis, mapping, and privacy constraints into a single product aimed at improving how outreach interview data is actually used.
